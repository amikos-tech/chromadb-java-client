    **BM25Tokenizer.java** (per D-03):
    Package `tech.amikos.chromadb.embeddings.bm25`. Public class.
    - Fields: `Set<String> stopWords`, `int tokenMaxLength` (default 100), `EnglishStemmer stemmer` (from `org.tartarus.snowball.ext.EnglishStemmer`).
    - Constructor: `BM25Tokenizer()` uses defaults. `BM25Tokenizer(Set<String> stopWords, int tokenMaxLength)` for custom.
    - `public List<String> tokenize(String text)`:
      1. `text.toLowerCase(Locale.ROOT)`
      2. `text.replaceAll("[^a-zA-Z0-9]+", " ")`
      3. `text.trim().split("\\s+")`  (handle empty string edge case — return empty list)
      4. Filter out tokens in `stopWords`
      5. Filter out tokens where `token.length() > tokenMaxLength`
      6. Stem each token: `stemmer.setCurrent(token); stemmer.stem(); stemmer.getCurrent()`
      7. Return list of stemmed tokens (not deduplicated — TF counting happens in BM25)

    **BM25EmbeddingFunction.java** (per D-02, D-03):
    Package `tech.amikos.chromadb.embeddings.bm25`. Implements `SparseEmbeddingFunction`.
    - Constants: `float K = 1.2f`, `float B = 0.75f`, `float DEFAULT_AVG_DOC_LEN = 256.0f`.
    - Fields: `BM25Tokenizer tokenizer`, `float avgDocLen`.
    - Constructor: `BM25EmbeddingFunction()` uses defaults. `BM25EmbeddingFunction(BM25Tokenizer tokenizer, float avgDocLen)` for custom.
    - `embedQuery(String query)`: Delegates to `embedSingle(query)`.
    - `embedDocuments(List<String> documents)`: Maps each doc to `embedSingle(doc)`.
    - `private SparseVector embedSingle(String text)`:
      1. `List<String> tokens = tokenizer.tokenize(text)`
      2. `int docLen = tokens.size()`
      3. Count term frequencies: `Map<String, Integer> tf = new LinkedHashMap<>()`
      4. For each token, compute BM25 score: `score = (tfVal * (K + 1)) / (tfVal + K * (1 - B + B * docLen / avgDocLen))`
      5. Hash stemmed token to int index: `int idx = Murmur3.hash32(token.getBytes(StandardCharsets.UTF_8), 0)`
      6. Accumulate scores per index (collision: sum)
      7. Sort indices ascending
      8. Build `int[]` indices and `float[]` values arrays
      9. Return `SparseVector.of(indices, values)`
    - Handle empty text: return `SparseVector.of(new int[0], new float[0])`.
